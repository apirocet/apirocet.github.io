iarray.numpy2iarray¶

iarray.numpy2iarray(arr: numpy.ndarray, cfg: Optional[iarray.config_params.Config] = None, **kwargs) → iarray.iarray_container.IArray¶

Convert a NumPy array into an ironArray array.

cfg and kwargs are the same than for empty().

Parameters

arr (np.ndarray) – The array to convert.

Returns

The new ironArray array.
