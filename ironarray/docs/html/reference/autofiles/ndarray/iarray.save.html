iarray.save¶

iarray.save(urlpath: str, iarr: iarray.iarray_container.IArray, cfg: Optional[iarray.config_params.Config] = None, **kwargs) → None¶

Save an array to a binary file in ironArray .iarray format.

cfg and kwargs are the same than for empty().

Parameters

  • iarr (IArray) – The array to save.

  • urlpath (str) – The url path to save the array.
