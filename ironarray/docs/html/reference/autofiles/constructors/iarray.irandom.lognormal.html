iarray.irandom.lognormal¶

iarray.irandom.lognormal(dtshape: iarray.constructors.DTShape, mean: float = 0.0, sigma: float = 1.0, cfg: Optional[iarray.config_params.Config] = None, **kwargs) → iarray.iarray_container.IArray¶

Draw samples from a log-normal distribution.

The cfg and kwargs parameters are the same than in random.random_sample().

Parameters

  • dtshape (DTShape) – The shape and data type of the array to be created.

  • mean (float or array_like of floats, optional) – Mean value of the underlying normal distribution. Default is 0.

  • sigma (float or array_like of floats, optional) – Standard deviation of the underlying normal distribution. Must be non-negative. Default is 1.

Returns

The new array.

Return type

IArray

See also

random.random_sample, np.random.lognormal
